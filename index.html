sif is a blazing-fast pentesting (recon/exploitation) suite written in Go 🐾

⚠️ currently in active development

Features

  • 📂 Directory/File fuzzing/scanning
  • 📡 DNS subdomains enumeration
  • 🐾 Common Web scanning
  • 🖥️ Port/Service scanning
  • 🦠 Vulnerability scanner
    • Support for pre-existing nuclei templates
    • Metasploit emulation for execution
  • 🔎 Automated google dorking
  • 💘 Shodan integration
